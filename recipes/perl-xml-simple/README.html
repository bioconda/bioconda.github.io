2.25-1, 2.25-0, 2.22-3, 2.22-2, 2.22-1, 2.22-0

  • perl-bio-automatedannotation

Installation

With an activated Bioconda channel (see 2. Set up channels), install with:

conda install perl-xml-simple

and update with:

conda update perl-xml-simple

or use the docker container:

docker pull quay.io/biocontainers/perl-xml-simple:<tag>

(see perl-xml-simple/tags for valid values for <tag>)
