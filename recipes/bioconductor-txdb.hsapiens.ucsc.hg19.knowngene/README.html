Installation

With an activated Bioconda channel (see 2. Set up channels), install with:

conda install bioconductor-txdb.hsapiens.ucsc.hg19.knowngene

and update with:

conda update bioconductor-txdb.hsapiens.ucsc.hg19.knowngene

or use the docker container:

docker pull quay.io/biocontainers/bioconductor-txdb.hsapiens.ucsc.hg19.knowngene:<tag>

(see bioconductor-txdb.hsapiens.ucsc.hg19.knowngene/tags for valid values for <tag>)
