coreutils¶

The GNU Core Utilities are the basic file, shell and text manipulation utilities of the GNU operating system. These are the core utilities which are expected to exist on every operating system.

Home http://www.gnu.org/software/coreutils/
Versions 8.25
License GPL
Recipe https://github.com/bioconda/bioconda-recipes/tree/master/recipes/coreutils

Installation¶

With an activated Bioconda channel (see 2. Set up channels), install with:

conda install coreutils

and update with:

conda update coreutils

A Docker container is available at https://quay.io/repository/biocontainers/coreutils.
