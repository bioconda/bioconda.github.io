Installation

With an activated Bioconda channel (see set-up-channels), install with:

conda install seqtk

and update with:

conda update seqtk

or use the docker container:

docker pull quay.io/biocontainers/seqtk:<tag>

(see seqtk/tags for valid values for <tag>)
