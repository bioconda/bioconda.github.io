package bioconductor-browserviz¶

Versions

2.4.0-0, 2.2.0-0, 1.10.0-0

Depends

  • bioconductor-biocgenerics >=0.28.0,<0.29.0

  • r-base >=3.5.1,<3.5.2.0a0

  • r-httpuv >=1.4.0

  • r-jsonlite >=1.5

Required By

  • bioconductor-browservizdemo

  • bioconductor-igvr

  • bioconductor-rcyjs

Installation

With an activated Bioconda channel (see 2. Set up channels), install with:

conda install bioconductor-browserviz

and update with:

conda update bioconductor-browserviz

or use the docker container:

docker pull quay.io/biocontainers/bioconductor-browserviz:<tag>

(see bioconductor-browserviz/tags for valid values for <tag>)
