package perl-bio-samtools¶

Versions:

1.43-3,  1.43-2,  1.43-1,  1.43-0

Depends:

  • libgcc-ng >=10.3.0

  • libzlib >=1.2.11,<1.3.0a0

  • perl >=5.32.1,<5.33.0a0 *_perl5

  • perl-bioperl-core 1.7.8.*

  • zlib >=1.2.11,<1.3.0a0

Required By:

  • perl-bioperl-run

  • perl-pcap

  • perl-sanger-cgp-allelecount

  • perl-bioperl

  • rad_haplotyper

  • perl-bio-viennangs

Installation

With an activated Bioconda channel (see set-up-channels), install with:

conda install perl-bio-samtools

and update with:

conda update perl-bio-samtools

or use the docker container:

docker pull quay.io/biocontainers/perl-bio-samtools:<tag>

(see perl-bio-samtools/tags for valid values for <tag>)
