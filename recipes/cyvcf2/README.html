Installation

With an activated Bioconda channel (see set-up-channels), install with:

conda install cyvcf2

and update with:

conda update cyvcf2

or use the docker container:

docker pull quay.io/biocontainers/cyvcf2:<tag>

(see cyvcf2/tags for valid values for <tag>)
