r-amap
======

Tools for Clustering and Principal Component Analysis (With robust methods, and parallelized functions).

======== ===========
Home     http://mulcyber.toulouse.inra.fr/projects/amap/
Versions 0.8_14
License  GPL
Recipe   https://github.com/bioconda/bioconda-recipes/tree/master/recipes/r-amap
======== ===========

Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install r-amap

and update with::

   conda update r-amap



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/r-amap.
