.. _`bioconductor-annotationforge`:

bioconductor-annotationforge
============================

|downloads|

Provides code for generating Annotation packages and their databases.  Packages produced are intended to be used with AnnotationDbi.

======== ===========
Home     http://bioconductor.org/packages/release/bioc/html/AnnotationForge.html
Versions 1.12.2, 1.14.0, 1.14.2
License  Artistic-2.0
Recipe   https://github.com/bioconda/bioconda-recipes/tree/master/recipes/bioconductor-annotationforge
======== ===========

Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install bioconductor-annotationforge

and update with::

   conda update bioconductor-annotationforge



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/bioconductor-annotationforge.
