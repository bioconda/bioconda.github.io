python-hivclustering
====================

|downloads|

A Python 3 library that makes inferences on HIV-1 transmission networks.

======== ===========
Home     https://github.com/veg/hivclustering
Versions 1.2.3
License  MIT
Recipe   https://github.com/bioconda/bioconda-recipes/tree/master/recipes/python-hivclustering
======== ===========

Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install python-hivclustering

and update with::

   conda update python-hivclustering



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/python-hivclustering.
