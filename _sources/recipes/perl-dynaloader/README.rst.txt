perl-dynaloader
===============

|downloads|

Dynamically load C libraries into Perl code

======== ===========
Home     http://metacpan.org/pod/DynaLoader
Versions 1.25
License  perl_5
Recipe   https://github.com/bioconda/bioconda-recipes/tree/master/recipes/perl-dynaloader/1.25
======== ===========

Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install perl-dynaloader

and update with::

   conda update perl-dynaloader



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/perl-dynaloader.
