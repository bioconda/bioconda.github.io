.. _`asciigenome`:

asciigenome
===========

|downloads|

Command-line genome browser running from terminal window and solely based on ASCII characters

======== ===========
Home     https://github.com/dariober/ASCIIGenome
Versions 0.2.0
License  MIT
Recipe   https://github.com/bioconda/bioconda-recipes/tree/master/recipes/asciigenome
======== ===========

Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install asciigenome

and update with::

   conda update asciigenome



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/asciigenome.
