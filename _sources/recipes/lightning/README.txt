lightning
=========

|downloads|

lightning is a library for large-scale linear classification, regression and ranking in Python

======== ===========
Home     http://contrib.scikit-learn.org/lightning/
Versions 0.2.dev0
License  Unknown
Recipe   https://github.com/bioconda/bioconda-recipes/tree/master/recipes/lightning
======== ===========

Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install lightning

and update with::

   conda update lightning



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/lightning.
