r-relations
===========

|downloads|

Data structures and algorithms for k-ary relations with arbitrary domains, featuring relational algebra, predicate functions, and fitters for consensus relations.

======== ===========
Home     
Versions 0.6_6
License  GPL-2
Recipe   https://github.com/bioconda/bioconda-recipes/tree/master/recipes/r-relations
======== ===========

Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install r-relations

and update with::

   conda update r-relations



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/r-relations.
