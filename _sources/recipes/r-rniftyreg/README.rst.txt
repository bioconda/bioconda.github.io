.. _`r-rniftyreg`:

r-rniftyreg
===========

|downloads|

Provides an R interface to the NiftyReg image registration tools <http://sourceforge.net/projects/niftyreg/>. Linear and nonlinear registration are supported, in two and three dimensions.

======== ===========
Home     https://github.com/jonclayden/RNiftyReg
Versions 2.2.0
License  GPL-2
Recipe   https://github.com/bioconda/bioconda-recipes/tree/master/recipes/r-rniftyreg
======== ===========

Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install r-rniftyreg

and update with::

   conda update r-rniftyreg



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/r-rniftyreg.
