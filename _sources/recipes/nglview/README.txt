.. _`nglview`:

nglview
=======

|downloads|

An IPython widget to interactively view molecular structures and trajectories. Utilizes the embeddable NGL Viewer for rendering.

======== ===========
Home     https://github.com/arose/nglview
Versions 0.5.1, 0.5.2, 0.6.1, 0.6.2.1
License  MIT
Recipe   https://github.com/bioconda/bioconda-recipes/tree/master/recipes/nglview
======== ===========

Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install nglview

and update with::

   conda update nglview



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/nglview.
