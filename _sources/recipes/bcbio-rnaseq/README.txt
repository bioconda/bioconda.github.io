bcbio-rnaseq
============

|downloads|

Report generation for bcbio-nextgen RNA-seq runs

======== ===========
Home     https://github.com/roryk/bcbio.rnaseq
Versions 1.0.3, 1.0.4, 1.1.1, 1.2.0
License  MIT
Recipe   https://github.com/bioconda/bioconda-recipes/tree/master/recipes/bcbio-rnaseq
======== ===========

Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install bcbio-rnaseq

and update with::

   conda update bcbio-rnaseq



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/bcbio-rnaseq.
