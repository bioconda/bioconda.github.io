.. _`bioconductor-seqpattern`:

bioconductor-seqpattern
=======================

|downloads|

Visualising oligonucleotide patterns and sequence motifs occurrences across a large set of sequences centred at a common reference point and sorted by a user defined feature.

======== ===========
Home     http://bioconductor.org/packages/release/bioc/html/seqPattern.html
Versions 1.2.0, 1.4.0, 1.6.0
License  GPL-3
Recipe   https://github.com/bioconda/bioconda-recipes/tree/master/recipes/bioconductor-seqpattern
======== ===========

Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install bioconductor-seqpattern

and update with::

   conda update bioconductor-seqpattern



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/bioconductor-seqpattern.
