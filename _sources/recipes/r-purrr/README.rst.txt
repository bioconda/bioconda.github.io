r-purrr
=======

|downloads|

Make your pure functions purr with the 'purrr' package. This package completes R's functional programming tools with missing features present in other programming languages.

======== ===========
Home     https://github.com/hadley/purrr
Versions 0.2.2
License  GPL-3 | file LICENSE
Recipe   https://github.com/bioconda/bioconda-recipes/tree/master/recipes/r-purrr
======== ===========

Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install r-purrr

and update with::

   conda update r-purrr



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/r-purrr.
