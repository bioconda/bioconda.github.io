razers3
=======

|downloads|

a tool for mapping millions of short genomic reads onto a reference genome

======== ===========
Home     http://www.seqan.de/projects/razers/
Versions 3.5.0
License  GPLv3
Recipe   https://github.com/bioconda/bioconda-recipes/tree/master/recipes/razers3
======== ===========

Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install razers3

and update with::

   conda update razers3



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/razers3.
