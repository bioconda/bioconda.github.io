hicexplorer
===========

|downloads|

Set of programs to process, analyze and visualize Hi-C data

======== ===========
Home     https://github.com/maxplanck-ie/HiCExplorer
Versions 0.1, 1.1a, 1.2, 1.3
License  GPL3
Recipe   https://github.com/bioconda/bioconda-recipes/tree/master/recipes/hicexplorer
======== ===========

Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install hicexplorer

and update with::

   conda update hicexplorer



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/hicexplorer.
