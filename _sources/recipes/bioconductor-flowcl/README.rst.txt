bioconductor-flowcl
===================

|downloads|

Semantic labelling of flow cytometric cell populations.

======== ===========
Home     http://bioconductor.org/packages/release/bioc/html/flowCL.html
Versions 1.10.0
License  Artistic-2.0
Recipe   https://github.com/bioconda/bioconda-recipes/tree/master/recipes/bioconductor-flowcl
======== ===========

Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install bioconductor-flowcl

and update with::

   conda update bioconductor-flowcl



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/bioconductor-flowcl.
