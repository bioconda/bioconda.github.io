Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install sistr_cmd

and update with::

   conda update sistr_cmd

A Docker container is available at https://quay.io/repository/biocontainers/sistr_cmd.
