Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install bcbio-nextgen

and update with::

   conda update bcbio-nextgen



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/bcbio-nextgen.
