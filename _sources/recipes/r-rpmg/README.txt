r-rpmg
======

|downloads|

Really Poor Man's Graphical User Interface, used to create interactive R analysis sessions with simple R commands.

======== ===========
Home     
Versions 2.2_1
License  GPL (>= 2)
Recipe   https://github.com/bioconda/bioconda-recipes/tree/master/recipes/r-rpmg
======== ===========

Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install r-rpmg

and update with::

   conda update r-rpmg



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/r-rpmg.
