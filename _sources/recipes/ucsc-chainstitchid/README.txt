Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install ucsc-chainstitchid

and update with::

   conda update ucsc-chainstitchid



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/ucsc-chainstitchid.
