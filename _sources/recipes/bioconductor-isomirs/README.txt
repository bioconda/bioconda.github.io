Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install bioconductor-isomirs

and update with::

   conda update bioconductor-isomirs



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/bioconductor-isomirs.
