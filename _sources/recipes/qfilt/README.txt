qfilt
=====

|downloads|

Filter sequencing data using some simple heuristics

======== ===========
Home     https://github.com/veg/qfilt
Versions 0.0.1
License  MIT
Recipe   https://github.com/bioconda/bioconda-recipes/tree/master/recipes/qfilt
======== ===========

Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install qfilt

and update with::

   conda update qfilt



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/qfilt.
