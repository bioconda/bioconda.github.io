ftputil
=======

|downloads|

High-level FTP client library (virtual file system and more)

======== ===========
Home     http://ftputil.sschwarzer.net/
Versions 3.2
License  BSD License
Recipe   https://github.com/bioconda/bioconda-recipes/tree/master/recipes/ftputil
======== ===========

Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install ftputil

and update with::

   conda update ftputil



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/ftputil.
