flask-sqlalchemy
================

|downloads|

Adds SQLAlchemy support to your Flask application

======== ===========
Home     http://github.com/mitsuhiko/flask-sqlalchemy
Versions 2.1
License  BSD License
Recipe   https://github.com/bioconda/bioconda-recipes/tree/master/recipes/flask-sqlalchemy
======== ===========

Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install flask-sqlalchemy

and update with::

   conda update flask-sqlalchemy



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/flask-sqlalchemy.
