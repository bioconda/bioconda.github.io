ucsc-faonerecord

Extract a single record from a .FA file

======== ===========
Home     http://hgdownload.cse.ucsc.edu/admin/exe/
Versions 332
License  varies; see http://genome.ucsc.edu/license
Recipe   https://github.com/bioconda/bioconda-recipes/tree/master/recipes/ucsc-faonerecord

Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install ucsc-faonerecord

and update with::

   conda update ucsc-faonerecord

A Docker container is available at https://quay.io/repository/biocontainers/ucsc-faonerecord.
