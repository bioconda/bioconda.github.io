.. _`repeatmasker`:

repeatmasker
============

|downloads|

RepeatMasker is a program that screens DNA sequences for interspersed repeats and low complexity DNA sequences.

======== ===========
Home     http://www.repeatmasker.org
Versions 4.0.6
License  Open Software License v2.1
Recipe   https://github.com/bioconda/bioconda-recipes/tree/master/recipes/repeatmasker
======== ===========

Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install repeatmasker

and update with::

   conda update repeatmasker



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/repeatmasker.
