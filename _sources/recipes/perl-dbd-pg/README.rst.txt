PostgreSQL database driver for the DBI module

======== ===========
Home     http://search.cpan.org/dist/DBD-Pg/
Versions 3.5.3
License  perl_5
Recipe   https://github.com/bioconda/bioconda-recipes/tree/master/recipes/perl-dbd-pg
======== ===========

Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install perl-dbd-pg

and update with::

   conda update perl-dbd-pg



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/perl-dbd-pg.
