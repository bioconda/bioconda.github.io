Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install pyfastaq

and update with::

   conda update pyfastaq



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/pyfastaq.
