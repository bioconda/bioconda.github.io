Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install rainbow

and update with::

   conda update rainbow



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/rainbow.
