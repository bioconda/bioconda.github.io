.. _`r-rseis`:

r-rseis
=======

|downloads|

Multiple interactive codes to view and analyze seismic data, via spectrum analysis, wavelet transforms, particle motion, hodograms.  Includes general time-series tools, plotting, filtering, interactive display.

======== ===========
Home     
Versions 3.5_2
License  GPL (>= 2)
Recipe   https://github.com/bioconda/bioconda-recipes/tree/master/recipes/r-rseis
======== ===========

Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install r-rseis

and update with::

   conda update r-rseis



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/r-rseis.
