igblast
=======

|downloads|

A tool for analyzing immunoglobulin (IG) and T cell receptor (TR) sequences.

======== ===========
Home     http://www.ncbi.nlm.nih.gov/projects/igblast/
Versions 1.4.0, 1.5.0
License  Public Domain
Recipe   https://github.com/bioconda/bioconda-recipes/tree/master/recipes/igblast
======== ===========

Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install igblast

and update with::

   conda update igblast



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/igblast.
