Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install age-metasv

and update with::

   conda update age-metasv



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/age-metasv.
