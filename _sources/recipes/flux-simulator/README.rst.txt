flux-simulator
==============

|downloads|

Tool for modeling RNA-Seq experiments in silico

======== ===========
Home     http://sammeth.net/confluence/display/SIM/Home
Versions 1.2.1
License  BSD
Recipe   https://github.com/bioconda/bioconda-recipes/tree/master/recipes/flux-simulator
======== ===========

Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install flux-simulator

and update with::

   conda update flux-simulator



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/flux-simulator.
