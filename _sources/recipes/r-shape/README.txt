r-shape
=======

|downloads|

Functions for plotting graphical shapes such as ellipses, circles, cylinders, arrows, ...

======== ===========
Home     
Versions 1.4.2
License  GPL (>= 3)
Recipe   https://github.com/bioconda/bioconda-recipes/tree/master/recipes/r-shape
======== ===========

Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install r-shape

and update with::

   conda update r-shape



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/r-shape.
