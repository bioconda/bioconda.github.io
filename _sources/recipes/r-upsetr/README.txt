.. _`r-upsetr`:

r-upsetr
========

|downloads|

Creates visualizations of intersecting sets using a novel matrix design, along with visualizations of several common set, element and attribute related tasks.

======== ===========
Home     http://github.com/hms-dbmi/UpSetR
Versions 1.0.3
License  MIT + file LICENSE
Recipe   https://github.com/bioconda/bioconda-recipes/tree/master/recipes/r-upsetr
======== ===========

Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install r-upsetr

and update with::

   conda update r-upsetr



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/r-upsetr.
