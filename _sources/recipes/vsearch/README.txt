Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install vsearch

and update with::

   conda update vsearch



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/vsearch.
