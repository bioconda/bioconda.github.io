r-nanostringnorm
================

|downloads|

A set of tools for normalizing, diagnostics and visualization of NanoString nCounter data.

======== ===========
Home     
Versions 1.1.21
License  GPL-2
Recipe   https://github.com/bioconda/bioconda-recipes/tree/master/recipes/r-nanostringnorm
======== ===========

Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install r-nanostringnorm

and update with::

   conda update r-nanostringnorm



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/r-nanostringnorm.
