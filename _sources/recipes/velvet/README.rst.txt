velvet

Sequence Assembler for short reads

======== ===========
Home     https://www.ebi.ac.uk/~zerbino/velvet/
Versions 1.2.10
License  GPL
Recipe   https://github.com/bioconda/bioconda-recipes/tree/master/recipes/velvet
======== ===========

Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install velvet

and update with::

   conda update velvet



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/velvet.
