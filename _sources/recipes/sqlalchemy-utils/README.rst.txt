sqlalchemy-utils
================

Various utility functions for SQLAlchemy.

License  BSD License

Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install sqlalchemy-utils

and update with::

   conda update sqlalchemy-utils



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/sqlalchemy-utils.
