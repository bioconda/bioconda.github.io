vardict-java
============

Home     https://github.com/AstraZeneca-NGS/VarDictJava
Versions 1.4.2, 1.4.3, 1.4.5, 1.4.6, 1.4.7
License  MIT
Recipe   https://github.com/bioconda/bioconda-recipes/tree/master/recipes/vardict-java
======== ===========

Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install vardict-java

and update with::

   conda update vardict-java



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/vardict-java.
