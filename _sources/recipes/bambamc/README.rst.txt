bambamc
=======

|downloads|

lightweight C implementation of name collating BAM file input and BAM file output

======== ===========
Home     https://github.com/gt1/bambamc
Versions 0.0.50
License  GPLv3
Recipe   https://github.com/bioconda/bioconda-recipes/tree/master/recipes/bambamc
======== ===========

Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install bambamc

and update with::

   conda update bambamc



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/bambamc.
