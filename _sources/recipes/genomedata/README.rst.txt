genomedata
==========

|downloads|

tools for accessing large amounts of genomic data

======== ===========
Home     http://pmgenomics.ca/hoffmanlab/proj/genomedata/
Versions 1.3.5
License  GPL2
Recipe   https://github.com/bioconda/bioconda-recipes/tree/master/recipes/genomedata
======== ===========

Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install genomedata

and update with::

   conda update genomedata



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/genomedata.
