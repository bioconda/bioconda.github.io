perl-capture-tiny
=================

|downloads|

Capture STDOUT and STDERR from Perl, XS or external programs

======== ===========
Home     https://github.com/dagolden/Capture-Tiny
Versions 0.34, 0.36
License  apache_2_0
Recipe   https://github.com/bioconda/bioconda-recipes/tree/master/recipes/perl-capture-tiny
======== ===========

Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install perl-capture-tiny

and update with::

   conda update perl-capture-tiny



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/perl-capture-tiny.
