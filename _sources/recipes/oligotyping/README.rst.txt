oligotyping

The oligotyping and minimum entropy decomposition (MED) pipeline for the analysis of marker gene amplicons

Versions 2.0, 2.1
License  GNU General Public License v3 or later (GPLv3+)

======== ===========

Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install oligotyping

and update with::

   conda update oligotyping



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/oligotyping.
