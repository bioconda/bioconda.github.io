ucsc-fafilter
=============

|downloads|

Filter fa records, selecting ones that match the specified conditions

======== ===========
Home     http://hgdownload.cse.ucsc.edu/admin/exe/
Versions 332
License  varies; see http://genome.ucsc.edu/license
Recipe   https://github.com/bioconda/bioconda-recipes/tree/master/recipes/ucsc-fafilter
======== ===========

Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install ucsc-fafilter

and update with::

   conda update ucsc-fafilter



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/ucsc-fafilter.
