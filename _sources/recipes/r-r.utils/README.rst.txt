r-r.utils
=========

|downloads|

Utility functions useful when programming and developing R packages.

======== ===========
Home     https://github.com/HenrikBengtsson/R.utils
Versions 2.2.0, 2.5.0
License  LGPL (>= 2.1)
Recipe   https://github.com/bioconda/bioconda-recipes/tree/master/recipes/r-r.utils
======== ===========

Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install r-r.utils

and update with::

   conda update r-r.utils



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/r-r.utils.
