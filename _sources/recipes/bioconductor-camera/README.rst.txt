bioconductor-camera
===================

|downloads|

Annotation of peaklists generated by xcms\, rule based annotation of isotopes and adducts\, isotope validation\, EIC correlation based tagging of unknown adducts and fragments

============= ===========
Home          https://bioconductor.org/packages/3.8/bioc/html/CAMERA.html
Versions      1.34.0, 1.32.0, 1.28.0, 1.26.0, 1.22.0
License       GPL (>= 2)
Recipe        https://github.com/bioconda/bioconda-recipes/tree/master/recipes//bioconductor-camera/meta.yaml



Links         biotools: :biotools:`camera`

============= ===========



Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install bioconductor-camera

and update with::

   conda update bioconductor-camera



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/bioconductor-camera.
