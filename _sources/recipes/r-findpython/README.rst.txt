r-findpython
============

|downloads|

Package designed to find an acceptable python binary.

======== ===========
Home     https://github.com/trevorld/findpython
Versions 1.0.1
License  MIT + file LICENSE
Recipe   https://github.com/bioconda/bioconda-recipes/tree/master/recipes/r-findpython
======== ===========

Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install r-findpython

and update with::

   conda update r-findpython



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/r-findpython.
