stacks
======

|downloads|

Stacks is a software pipeline for building loci from RAD-seq

======== ===========
Home     http://catchenlab.life.illinois.edu/stacks/
Versions 1.37, 1.40, 1.42
License  GPL
Recipe   https://github.com/bioconda/bioconda-recipes/tree/master/recipes/stacks
======== ===========

Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install stacks

and update with::

   conda update stacks



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/stacks.
