Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install ococo

and update with::

   conda update ococo



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/ococo.
