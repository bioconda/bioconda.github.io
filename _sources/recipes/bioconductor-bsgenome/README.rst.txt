bioconductor-bsgenome
=====================

|downloads|

Infrastructure shared by all the Biostrings-based genome data packages

======== ===========
Home     http://bioconductor.org/packages/release/bioc/html/BSgenome.html
Versions 1.38.0, 1.40.1, 1.42.0
License  Artistic-2.0
Recipe   https://github.com/bioconda/bioconda-recipes/tree/master/recipes/bioconductor-bsgenome
======== ===========

Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install bioconductor-bsgenome

and update with::

   conda update bioconductor-bsgenome



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/bioconductor-bsgenome.
