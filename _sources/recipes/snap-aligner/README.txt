snap-aligner
============

|downloads|

Scalable Nucleotide Alignment Program -- a fast and accurate read aligner for high-throughput sequencing data

======== ===========
Home     http://snap.cs.berkeley.edu/
Versions 1.0beta.18, 1.0beta.23, 1.0dev.97
License  Apache v2
Recipe   https://github.com/bioconda/bioconda-recipes/tree/master/recipes/snap-aligner
======== ===========

Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install snap-aligner

and update with::

   conda update snap-aligner



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/snap-aligner.
