Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install metilene

and update with::

   conda update metilene



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/metilene.
