Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install bd2k-python-lib

and update with::

   conda update bd2k-python-lib



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/bd2k-python-lib.
