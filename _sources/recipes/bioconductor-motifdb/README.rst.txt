bioconductor-motifdb
====================

|downloads|

More than 2000 annotated position frequency matrices from nine public sources, for multiple organisms.

======== ===========
Home     http://bioconductor.org/packages/release/bioc/html/MotifDb.html
Versions 1.16.0
License  Artistic-2.0 | file LICENSE
Recipe   https://github.com/bioconda/bioconda-recipes/tree/master/recipes/bioconductor-motifdb
======== ===========

Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install bioconductor-motifdb

and update with::

   conda update bioconductor-motifdb



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/bioconductor-motifdb.
