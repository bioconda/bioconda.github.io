bioconductor-go.db
==================

|downloads|

A set of annotation maps describing the entire Gene Ontology assembled using data from GO

======== ===========
Home     http://bioconductor.org/packages/release/data/annotation/html/GO.db.html
Versions 3.2.2, 3.3.0
License  Artistic-2.0
Recipe   https://github.com/bioconda/bioconda-recipes/tree/master/recipes/bioconductor-go.db
======== ===========

Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install bioconductor-go.db

and update with::

   conda update bioconductor-go.db



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/bioconductor-go.db.
