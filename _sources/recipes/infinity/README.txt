infinity
========

|downloads|

All-in-one infinity value for Python. Can be compared to any object.

======== ===========
Home     https://github.com/kvesteri/infinity
Versions 1.3
License  BSD License
Recipe   https://github.com/bioconda/bioconda-recipes/tree/master/recipes/infinity
======== ===========

Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install infinity

and update with::

   conda update infinity



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/infinity.
