Sensitive detection of INDELs (INsertions and DELetions)

Home     http://scalpel.sourceforge.net/
Versions 0.5.1
License  MIT

Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install scalpel

and update with::

   conda update scalpel
