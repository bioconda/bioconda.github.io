svgwrite
========

|downloads|

A Python library to create SVG drawings.

======== ===========
Home     http://bitbucket.org/mozman/svgwrite
Versions 1.1.6
License  MIT License
Recipe   https://github.com/bioconda/bioconda-recipes/tree/master/recipes/svgwrite
======== ===========

Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install svgwrite

and update with::

   conda update svgwrite



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/svgwrite.
