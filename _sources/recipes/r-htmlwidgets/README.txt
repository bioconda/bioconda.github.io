r-htmlwidgets
=============

|downloads|

A framework for creating HTML widgets that render in various contexts including the R console, 'R Markdown' documents, and 'Shiny' web applications.

======== ===========
Home     https://github.com/ramnathv/htmlwidgets
Versions 0.5
License  MIT + file LICENSE
Recipe   https://github.com/bioconda/bioconda-recipes/tree/master/recipes/r-htmlwidgets
======== ===========

Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install r-htmlwidgets

and update with::

   conda update r-htmlwidgets



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/r-htmlwidgets.
