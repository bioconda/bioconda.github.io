geodl
=====

|downloads|

Dowload FASTQ files from GEO-NCBI with ease

======== ===========
Home     https://github.com/jduc/geoDL
Versions 1.0b1
License  GPL
Recipe   https://github.com/bioconda/bioconda-recipes/tree/master/recipes/geodl
======== ===========

Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install geodl

and update with::

   conda update geodl



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/geodl.
