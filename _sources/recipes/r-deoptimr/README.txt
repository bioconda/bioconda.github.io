.. _`r-deoptimr`:

r-deoptimr
==========

|downloads|

An implementation of a bespoke jDE variant of the Differential Evolution stochastic algorithm for global optimization of nonlinear programming problems.

======== ===========
Home     
Versions 1.0_6
License  GPL (>= 2)
Recipe   https://github.com/bioconda/bioconda-recipes/tree/master/recipes/r-deoptimr
======== ===========

Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install r-deoptimr

and update with::

   conda update r-deoptimr



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/r-deoptimr.
