pytest-mock
===========

|downloads|

Thin-wrapper around the mock package for easier use with py.test

======== ===========
Home     https://github.com/pytest-dev/pytest-mock/
Versions 1.1
License  MIT License
Recipe   https://github.com/bioconda/bioconda-recipes/tree/master/recipes/pytest-mock
======== ===========

Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install pytest-mock

and update with::

   conda update pytest-mock



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/pytest-mock.
