Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install vawk

and update with::

   conda update vawk



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/vawk.
