metasv
======

|downloads|

An accurate and integrative structural-variant caller for next generation sequencing

======== ===========
Home     https://github.com/bioinform/metasv
Versions 0.4.0
License  MIT
Recipe   https://github.com/bioconda/bioconda-recipes/tree/master/recipes/metasv
======== ===========

Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install metasv

and update with::

   conda update metasv



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/metasv.
