bioconductor-protgenerics
=========================

|downloads|

S4 generic functions needed by Bioconductor proteomics packages.

======== ===========
Home     http://bioconductor.org/packages/release/bioc/html/ProtGenerics.html
Versions 1.2.1, 1.4.0
License  Artistic-2.0
Recipe   https://github.com/bioconda/bioconda-recipes/tree/master/recipes/bioconductor-protgenerics
======== ===========

Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install bioconductor-protgenerics

and update with::

   conda update bioconductor-protgenerics



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/bioconductor-protgenerics.
