Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install easydev

and update with::

   conda update easydev



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/easydev.
