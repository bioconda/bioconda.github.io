libssh2
=======

|downloads|

libssh2 is a client-side C library implementing the SSH2 protocol

======== ===========
Home     https://www.libssh2.org/
Versions 1.7.0
License  Custom OSS
Recipe   https://github.com/bioconda/bioconda-recipes/tree/master/recipes/libssh2
======== ===========

Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install libssh2

and update with::

   conda update libssh2



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/libssh2.
