piranha
=======

|downloads|

Piranha is a peak-caller for CLIP- and RIP-Seq data.

======== ===========
Home     http://smithlabresearch.org/software/piranha/
Versions 1.2.1
License  GNU General Public License v3 (GPLv3)
Recipe   https://github.com/bioconda/bioconda-recipes/tree/master/recipes/piranha
======== ===========

Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install piranha

and update with::

   conda update piranha



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/piranha.
