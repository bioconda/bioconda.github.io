ucsc-chainsort
==============

|downloads|

Sort chains.  By default sorts by score.

======== ===========
Home     http://hgdownload.cse.ucsc.edu/admin/exe/
Versions 324, 332
License  varies; see http://genome.ucsc.edu/license
Recipe   https://github.com/bioconda/bioconda-recipes/tree/master/recipes/ucsc-chainsort
======== ===========

Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install ucsc-chainsort

and update with::

   conda update ucsc-chainsort



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/ucsc-chainsort.
