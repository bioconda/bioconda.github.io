.. _`r-venndiagram`:

r-venndiagram
=============

|downloads|

A set of functions to generate high-resolution Venn and Euler plots. Includes handling for several special cases, including two-case scaling, and extensive customization of plot shape and structure.

Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install r-venndiagram

and update with::

   conda update r-venndiagram

A Docker container is available at https://quay.io/repository/biocontainers/r-venndiagram.
