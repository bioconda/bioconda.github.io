r-ash
=====

|downloads|

David Scott's ASH routines ported from S-PLUS to R.

======== ===========
Home     
Versions 1.0_15
License  GPL (>= 2)
Recipe   https://github.com/bioconda/bioconda-recipes/tree/master/recipes/r-ash
======== ===========

Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install r-ash

and update with::

   conda update r-ash



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/r-ash.
