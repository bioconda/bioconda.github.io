.. _`r-compquadform`:

r-compquadform
==============

|downloads|

Computes the distribution function of quadratic forms in normal variables using Imhof's method, Davies's algorithm, Farebrother's algorithm or Liu et al.'s algorithm.

======== ===========
Home     
Versions 1.4.1
License  GPL (>= 2)
Recipe   https://github.com/bioconda/bioconda-recipes/tree/master/recipes/r-compquadform
======== ===========

Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install r-compquadform

and update with::

   conda update r-compquadform



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/r-compquadform.
