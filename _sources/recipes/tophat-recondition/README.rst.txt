Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install tophat-recondition

and update with::

   conda update tophat-recondition



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/tophat-recondition.
