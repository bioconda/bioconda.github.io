Basic Local Alignment and Search Tool for Oxford Nanopore Long Sequences

======== ===========
Home     https://github.com/ruhulsbu/NanoBLASTer
Versions 0.16
License  MIT
Recipe   https://github.com/bioconda/bioconda-recipes/tree/master/recipes/nanoblaster
======== ===========

Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install nanoblaster

and update with::

   conda update nanoblaster

A Docker container is available at https://quay.io/repository/biocontainers/nanoblaster.
