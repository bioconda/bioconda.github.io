License  BSD License

======== ===========

Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install flask-basicauth

and update with::

   conda update flask-basicauth



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/flask-basicauth.
