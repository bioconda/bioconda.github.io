bioconductor-rcytoscape
=======================

|downloads|

Interactvive viewing and exploration of graphs, connecting R to Cytoscape.

======== ===========
Home     http://bioconductor.org/packages/release/bioc/html/RCytoscape.html
Versions 1.21.1
License  GPL-2
Recipe   https://github.com/bioconda/bioconda-recipes/tree/master/recipes/bioconductor-rcytoscape
======== ===========

Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install bioconductor-rcytoscape

and update with::

   conda update bioconductor-rcytoscape



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/bioconductor-rcytoscape.
