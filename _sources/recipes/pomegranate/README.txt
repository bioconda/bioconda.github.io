pomegranate
===========

|downloads|

Pomegranate is a graphical models library for Python, implemented in Cython for speed.

======== ===========
Home     http://pypi.python.org/pypi/pomegranate/
Versions 0.3.7
License  LICENSE.txt
Recipe   https://github.com/bioconda/bioconda-recipes/tree/master/recipes/pomegranate
======== ===========

Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install pomegranate

and update with::

   conda update pomegranate



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/pomegranate.
