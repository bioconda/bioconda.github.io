Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install biom-format

and update with::

   conda update biom-format



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/biom-format.
