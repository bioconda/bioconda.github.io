galaxy-lib
==========

|downloads|

Subset of Galaxy (http://galaxyproject.org/) core code base designed to be used a library.

======== ===========
Home     https://github.com/galaxyproject/galaxy-lib
Versions 16.10.3, 16.10.4, 16.10.6, 16.10.8, 16.10.9, 16.4.0, 16.7.10
License  Academic Free License (AFL)
Recipe   https://github.com/bioconda/bioconda-recipes/tree/master/recipes/galaxy-lib
======== ===========

Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install galaxy-lib

and update with::

   conda update galaxy-lib



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/galaxy-lib.
