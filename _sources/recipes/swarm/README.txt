swarm
=====

|downloads|

A robust and fast clustering method for amplicon-based studies.

======== ===========
Home     https://github.com/torognes/swarm
Versions 1.2.19, 2.1.5
License  Affero GPL
Recipe   https://github.com/bioconda/bioconda-recipes/tree/master/recipes/swarm
======== ===========

Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install swarm

and update with::

   conda update swarm



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/swarm.
