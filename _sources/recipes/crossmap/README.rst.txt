Recipe   https://github.com/bioconda/bioconda-recipes/tree/master/recipes/crossmap

Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install crossmap

and update with::

   conda update crossmap

A Docker container is available at https://quay.io/repository/biocontainers/crossmap.
