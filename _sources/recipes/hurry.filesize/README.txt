hurry.filesize
==============

|downloads|

A simple Python library for human readable file sizes (or anything sized in bytes).

======== ===========
Home     
Versions 0.9
License  ZPL 2.1
Recipe   https://github.com/bioconda/bioconda-recipes/tree/master/recipes/hurry.filesize
======== ===========

Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install hurry.filesize

and update with::

   conda update hurry.filesize



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/hurry.filesize.
