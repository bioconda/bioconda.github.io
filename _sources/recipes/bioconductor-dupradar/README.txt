bioconductor-dupradar
=====================

|downloads|

Duplication rate quality control for RNA-Seq datasets.

======== ===========
Home     http://bioconductor.org/packages/release/bioc/html/dupRadar.html
Versions 1.2.2
License  GPL-3
Recipe   https://github.com/bioconda/bioconda-recipes/tree/master/recipes/bioconductor-dupradar
======== ===========

Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install bioconductor-dupradar

and update with::

   conda update bioconductor-dupradar



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/bioconductor-dupradar.
