edlib
=====

|downloads|

C/C++ library and program for sequence alignment using edit (Levenshtein) distance

======== ===========
Home     https://github.com/Martinsos/edlib/
Versions 1.0.0
License  MIT
Recipe   https://github.com/bioconda/bioconda-recipes/tree/master/recipes/edlib
======== ===========

Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install edlib

and update with::

   conda update edlib



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/edlib.
