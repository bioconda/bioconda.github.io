Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install python-dateutil

and update with::

   conda update python-dateutil



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/python-dateutil.
