eagle
=====

|downloads|

Eagle is a webtool for genome variants and snp analysis

======== ===========
Home     https://bitbucket.org/christopherschroeder/eagle
Versions 0.9.0
License  MIT License
Recipe   https://github.com/bioconda/bioconda-recipes/tree/master/recipes/eagle
======== ===========

Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install eagle

and update with::

   conda update eagle



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/eagle.
