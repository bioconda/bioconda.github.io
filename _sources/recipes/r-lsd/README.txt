r-lsd
=====

|downloads|

Create lots of colorful plots in a plethora of variations (try the LSD demotour() )

======== ===========
Home     
Versions 3.0
License  Unlimited
Recipe   https://github.com/bioconda/bioconda-recipes/tree/master/recipes/r-lsd
======== ===========

Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install r-lsd

and update with::

   conda update r-lsd



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/r-lsd.
