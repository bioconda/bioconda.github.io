License  unknown
Recipe   https://github.com/bioconda/bioconda-recipes/tree/master/recipes/perl-bloom-faster

Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install perl-bloom-faster

and update with::

   conda update perl-bloom-faster



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/perl-bloom-faster.
