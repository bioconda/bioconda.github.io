nose-capturestderr
==================

|downloads|

Nose plugin for capturing stderr.

======== ===========
Home     http://github.com/sio2project/nose-capturestderr
Versions 1.0
License  GPL
Recipe   https://github.com/bioconda/bioconda-recipes/tree/master/recipes/nose-capturestderr
======== ===========

Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install nose-capturestderr

and update with::

   conda update nose-capturestderr



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/nose-capturestderr.
