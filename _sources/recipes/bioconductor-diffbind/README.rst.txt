.. _`bioconductor-diffbind`:

bioconductor-diffbind
=====================

|downloads|

Compute differentially bound sites from multiple ChIP-seq experiments using affinity (quantitative) data. Also enables occupancy (overlap) analysis and plotting functions.

======== ===========
Home     http://bioconductor.org/packages/release/bioc/html/DiffBind.html
Versions 1.16.3
License  Artistic-2.0
Recipe   https://github.com/bioconda/bioconda-recipes/tree/master/recipes/bioconductor-diffbind
======== ===========

Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install bioconductor-diffbind

and update with::

   conda update bioconductor-diffbind



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/bioconductor-diffbind.
