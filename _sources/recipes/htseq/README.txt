htseq
=====

|downloads|

A framework to process and analyze data from high-throughput sequencing (HTS) assays

======== ===========
Home     http://www-huber.embl.de/users/anders/HTSeq/
Versions 0.6.1, 0.6.1.post1, 0.6.1p1
License  GNU General Public License (GPL)
Recipe   https://github.com/bioconda/bioconda-recipes/tree/master/recipes/htseq
======== ===========

Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install htseq

and update with::

   conda update htseq



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/htseq.
