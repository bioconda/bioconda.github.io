Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install yasm

and update with::

   conda update yasm



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/yasm.
