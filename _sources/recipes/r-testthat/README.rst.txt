r-testthat
==========

|downloads|

A unit testing system designed to be fun, flexible and easy to set up.

======== ===========
Home     https://github.com/hadley/testthat
Versions 1.0.2
License  MIT + file LICENSE
Recipe   https://github.com/bioconda/bioconda-recipes/tree/master/recipes/r-testthat
======== ===========

Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install r-testthat

and update with::

   conda update r-testthat



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/r-testthat.
