ruamel.yaml
===========

|downloads|

A YAML package for Python. It is a derivative of Kirill Simonov's PyYAML 3.11 which supports YAML1.1

======== ===========
Home     https://bitbucket.org/ruamel/yaml
Versions 0.11.11, 0.12.13
License  MIT
Recipe   https://github.com/bioconda/bioconda-recipes/tree/master/recipes/ruamel.yaml
======== ===========

Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install ruamel.yaml

and update with::

   conda update ruamel.yaml



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/ruamel.yaml.
