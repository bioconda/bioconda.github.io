urllib3
=======

|downloads|

HTTP library with thread-safe connection pooling, file post, and more.

======== ===========
Home     http://urllib3.readthedocs.org/
Versions 1.12
License  MIT License
Recipe   https://github.com/bioconda/bioconda-recipes/tree/master/recipes/urllib3
======== ===========

Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install urllib3

and update with::

   conda update urllib3



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/urllib3.
