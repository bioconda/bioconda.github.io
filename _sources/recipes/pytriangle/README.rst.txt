pytriangle
==========

|downloads|

A python interface to the 2D triangulation program TRIANGLE

======== ===========
Home     https://github.com/pletzer/pytriangle
Versions 1.0.0, 1.0.9
License  MIT
Recipe   https://github.com/bioconda/bioconda-recipes/tree/master/recipes/pytriangle
======== ===========

Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install pytriangle

and update with::

   conda update pytriangle



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/pytriangle.
