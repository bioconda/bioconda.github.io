.. _`r-longitudinal`:

r-longitudinal
==============

|downloads|

Contains general data structures and functions for longitudinal data with multiple variables, repeated measurements, and irregularly spaced time points. Also implements a shrinkage estimator of dynamical correlation and dynamical covariance.

======== ===========
Home     http://strimmerlab.org/software/genenet/
Versions 1.1.12
License  GPL (>= 3)
Recipe   https://github.com/bioconda/bioconda-recipes/tree/master/recipes/r-longitudinal
======== ===========

Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install r-longitudinal

and update with::

   conda update r-longitudinal



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/r-longitudinal.
