bioconductor-derfinderhelper
============================

|downloads|

Helper package for speeding up the derfinder package when using multiple cores.

============= ===========
Home          http://bioconductor.org/packages/3.6/bioc/html/derfinderHelper.html
Versions      1.10.0, 1.12.0
License       Artistic-2.0
Recipe        https://github.com/bioconda/bioconda-recipes/tree/master/recipes/bioconductor-derfinderhelper



============= ===========



Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install bioconductor-derfinderhelper

and update with::

   conda update bioconductor-derfinderhelper



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/bioconductor-derfinderhelper.
