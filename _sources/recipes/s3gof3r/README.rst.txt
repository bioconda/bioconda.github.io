Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install s3gof3r

and update with::

   conda update s3gof3r



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/s3gof3r.
