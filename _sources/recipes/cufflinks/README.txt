Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install cufflinks

and update with::

   conda update cufflinks



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/cufflinks.
