dendropy
========

|downloads|

A Python library for phylogenetics and phylogenetic computing: reading, writing, simulation, processing and manipulation of phylogenetic trees (phylogenies) and characters.

======== ===========
Home     http://packages.python.org/DendroPy/
Versions 3.12.3, 4.0.3
License  BSD License
Recipe   https://github.com/bioconda/bioconda-recipes/tree/master/recipes/dendropy
======== ===========

Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install dendropy

and update with::

   conda update dendropy



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/dendropy.
