Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install burrito-fillings

and update with::

   conda update burrito-fillings



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/burrito-fillings.
