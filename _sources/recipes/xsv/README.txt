xsv
===

A fast CSV toolkit written in Rust.

======== ===========
Home     https://github.com/BurntSushi/xsv
Versions 0.9.8
License  MIT
Recipe   https://github.com/bioconda/bioconda-recipes/tree/master/recipes/xsv
======== ===========

Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install xsv

and update with::

   conda update xsv



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/xsv.
