.. _`bioconductor-affy`:

bioconductor-affy
=================

|downloads|

The package contains functions for exploratory oligonucleotide array analysis. The dependence on tkWidgets only concerns few convenience functions. 'affy' is fully functional without it.

======== ===========
Home     http://bioconductor.org/packages/release/bioc/html/affy.html
Versions 1.48.0, 1.50.0
License  LGPL (>= 2.0)
Recipe   https://github.com/bioconda/bioconda-recipes/tree/master/recipes/bioconductor-affy
======== ===========

Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install bioconductor-affy

and update with::

   conda update bioconductor-affy



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/bioconductor-affy.
