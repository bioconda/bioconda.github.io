ucsc-subchar
============

|downloads|

Substitute one character for another throughout a file.

======== ===========
Home     http://hgdownload.cse.ucsc.edu/admin/exe/
Versions 324, 332
License  varies; see http://genome.ucsc.edu/license
Recipe   https://github.com/bioconda/bioconda-recipes/tree/master/recipes/ucsc-subchar
======== ===========

Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install ucsc-subchar

and update with::

   conda update ucsc-subchar



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/ucsc-subchar.
