.. _`bioconductor-flowdensity`:

bioconductor-flowdensity
========================

|downloads|

This package provides tools for automated sequential gating analogous to the manual gating strategy based on the density of the data.

======== ===========
Home     http://bioconductor.org/packages/release/bioc/html/flowDensity.html
Versions 1.6.0
License  Artistic-2.0
Recipe   https://github.com/bioconda/bioconda-recipes/tree/master/recipes/bioconductor-flowdensity
======== ===========

Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install bioconductor-flowdensity

and update with::

   conda update bioconductor-flowdensity



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/bioconductor-flowdensity.
