Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install msprime

and update with::

   conda update msprime



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/msprime.
