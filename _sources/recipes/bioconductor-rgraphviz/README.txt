.. _`bioconductor-rgraphviz`:

bioconductor-rgraphviz
======================

|downloads|

Interfaces R with the AT and T graphviz library for plotting R graph objects from the graph package.

======== ===========
Home     http://bioconductor.org/packages/release/bioc/html/Rgraphviz.html
Versions 2.13.0, 2.14.0, 2.16.0
License  EPL
Recipe   https://github.com/bioconda/bioconda-recipes/tree/master/recipes/bioconductor-rgraphviz
======== ===========

Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install bioconductor-rgraphviz

and update with::

   conda update bioconductor-rgraphviz



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/bioconductor-rgraphviz.
