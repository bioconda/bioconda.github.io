Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install biopython

and update with::

   conda update biopython



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/biopython.
