r-gmd
=====

|downloads|

GMD is a package for non-parametric distance measurement between two discrete frequency distributions.

======== ===========
Home     http://CRAN.R-project.org/package=GMD
Versions 0.3.3
License  GPL (>= 2)
Recipe   https://github.com/bioconda/bioconda-recipes/tree/master/recipes/r-gmd
======== ===========

Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install r-gmd

and update with::

   conda update r-gmd



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/r-gmd.
