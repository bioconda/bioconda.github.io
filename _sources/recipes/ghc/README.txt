Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install ghc

and update with::

   conda update ghc



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/ghc.
