vardict
=======

|downloads|

A sensitive variant caller for both single and paired sample variant calling

======== ===========
Home     https://github.com/AstraZeneca-NGS/VarDict
Versions 2015.10.30, 2016.02.19
License  MIT
Recipe   https://github.com/bioconda/bioconda-recipes/tree/master/recipes/vardict
======== ===========

Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install vardict

and update with::

   conda update vardict



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/vardict.
