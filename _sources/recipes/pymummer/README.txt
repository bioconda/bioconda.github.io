Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install pymummer

and update with::

   conda update pymummer



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/pymummer.
