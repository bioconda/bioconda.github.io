.. _`r-beanplot`:

r-beanplot
==========

|downloads|

Plots univariate comparison graphs, an alternative to boxplot/stripchart/violin plot.

======== ===========
Home     
Versions 1.2
License  GPL-2
Recipe   https://github.com/bioconda/bioconda-recipes/tree/master/recipes/r-beanplot
======== ===========

Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install r-beanplot

and update with::

   conda update r-beanplot



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/r-beanplot.
