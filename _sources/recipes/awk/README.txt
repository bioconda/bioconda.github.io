.. _`gawk`:

gawk
====

|downloads|

The awk utility interprets a special-purpose programming language that makes it possible to handle simple data-reformatting jobs with just a few lines of code.

======== ===========
Home     http://www.gnu.org/software/gawk/
Versions 4.1.0
License  GPL
Recipe   https://github.com/bioconda/bioconda-recipes/tree/master/recipes/awk
======== ===========

Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`setup`), install with::

   conda install gawk

and update with::

   conda update gawk



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/gawk.
