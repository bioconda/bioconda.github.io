kallisto
========

|downloads|

Quantifying abundances of transcripts from RNA-Seq data, or more generally of target sequences using high-throughput sequencing reads.

======== ===========
Home     http://pachterlab.github.io/kallisto
Versions 0.42.3, 0.42.4, 0.43.0
License  Custom
Recipe   https://github.com/bioconda/bioconda-recipes/tree/master/recipes/kallisto
======== ===========

Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install kallisto

and update with::

   conda update kallisto



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/kallisto.
