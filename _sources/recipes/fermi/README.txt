fermi
=====

A WGS de novo assembler based on the FMD-index for large genomes

======== ===========
Home     https://github.com/lh3/fermi
Versions 1.1_r751_beta
License  Unknown
Recipe   https://github.com/bioconda/bioconda-recipes/tree/master/recipes/fermi
======== ===========

Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install fermi

and update with::

   conda update fermi

A Docker container is available at https://quay.io/repository/biocontainers/fermi.
