======== ===========

Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install textinput

and update with::

   conda update textinput



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/textinput.
