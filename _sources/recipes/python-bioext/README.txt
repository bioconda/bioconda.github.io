python-bioext
=============

|downloads|

A few handy bioinformatics tools not already in BioPython

======== ===========
Home     https://github.com/veg/BioExt.git
Versions 0.17.4
License  GPL-3
Recipe   https://github.com/bioconda/bioconda-recipes/tree/master/recipes/python-bioext
======== ===========

Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install python-bioext

and update with::

   conda update python-bioext



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/python-bioext.
