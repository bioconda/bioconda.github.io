r-mutationalpatterns
====================

|downloads|

R package for extracting and visualizing mutational patterns in base substitution catalogues

======== ===========
Home     https://github.com/CuppenResearch/MutationalPatterns
Versions 0.2b
License  MIT
Recipe   https://github.com/bioconda/bioconda-recipes/tree/master/recipes/r-mutationalpatterns
======== ===========

Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install r-mutationalpatterns

and update with::

   conda update r-mutationalpatterns



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/r-mutationalpatterns.
