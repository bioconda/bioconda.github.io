r-gamlss.dist
=============

|downloads|

The different distributions used for the response variables in GAMLSS modelling.

======== ===========
Home     http://www.gamlss.org/
Versions 5.0_0
License  GPL-2 | GPL-3
Recipe   https://github.com/bioconda/bioconda-recipes/tree/master/recipes/r-gamlss.dist
======== ===========

Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install r-gamlss.dist

and update with::

   conda update r-gamlss.dist



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/r-gamlss.dist.
