metacluster
===========

|downloads|

MetaCluster5.1 is a new software for binning short pair-end reads

======== ===========
Home     http://i.cs.hku.hk/~alse/MetaCluster/
Versions 5.1
License  GPLv2
Recipe   https://github.com/bioconda/bioconda-recipes/tree/master/recipes/metacluster
======== ===========

Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install metacluster

and update with::

   conda update metacluster



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/metacluster.
