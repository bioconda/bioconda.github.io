sphinx-argparse
===============

|downloads|

Sphinx extension that automatically document argparse commands and options

======== ===========
Home     
Versions 0.1.15
License  MIT
Recipe   https://github.com/bioconda/bioconda-recipes/tree/master/recipes/sphinx-argparse
======== ===========

Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install sphinx-argparse

and update with::

   conda update sphinx-argparse



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/sphinx-argparse.
