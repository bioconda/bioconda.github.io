r-preseqr
=========

|downloads|

Estimating the number of species represented r or more times in a random sample.

======== ===========
Home     
Versions 2.0.0
License  GPL-3
Recipe   https://github.com/bioconda/bioconda-recipes/tree/master/recipes/r-preseqr
======== ===========

Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install r-preseqr

and update with::

   conda update r-preseqr



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/r-preseqr.
