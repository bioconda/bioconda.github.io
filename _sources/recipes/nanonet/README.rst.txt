nanonet
=======

|downloads|

Nanonet provides recurrent neural network basecalling for Oxford Nanopore MinION data.

======== ===========
Home     https://github.com/nanoporetech/nanonet
Versions 2.0.0
License  MPL-2.0
Recipe   https://github.com/bioconda/bioconda-recipes/tree/master/recipes/nanonet
======== ===========

Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install nanonet

and update with::

   conda update nanonet



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/nanonet.
