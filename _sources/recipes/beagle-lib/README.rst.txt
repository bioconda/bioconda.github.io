beagle-lib
==========

|downloads|

general purpose library for evaluating the likelihood of sequence evolution on trees

======== ===========
Home     https://github.com/beagle-dev/beagle-lib
Versions 2.1.2
License  GPLv3+
Recipe   https://github.com/bioconda/bioconda-recipes/tree/master/recipes/beagle-lib
======== ===========

Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install beagle-lib

and update with::

   conda update beagle-lib



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/beagle-lib.
