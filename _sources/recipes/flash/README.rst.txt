flash

|downloads|

Merge mates from fragments that are shorter than twice the read length

Home     https://ccb.jhu.edu/software/FLASH/
Versions 1.2.11
License  GPLv3+

======== ===========

Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install flash

and update with::

   conda update flash
