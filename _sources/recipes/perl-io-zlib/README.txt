perl-io-zlib

IO:: style interface to Compress::Zlib

======== ===========
Home     http://metacpan.org/pod/IO::Zlib
Versions 1.10
License  unknown
Recipe   https://github.com/bioconda/bioconda-recipes/tree/master/recipes/perl-io-zlib
======== ===========

Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install perl-io-zlib

and update with::

   conda update perl-io-zlib



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/perl-io-zlib.
