Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install lumpy-sv

and update with::

   conda update lumpy-sv

A Docker container is available at https://quay.io/repository/biocontainers/lumpy-sv.
