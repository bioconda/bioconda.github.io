Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install google-api-python-client

and update with::

   conda update google-api-python-client



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/google-api-python-client.
