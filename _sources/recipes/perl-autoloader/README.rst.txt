Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install perl-autoloader

and update with::

   conda update perl-autoloader

A Docker container is available at https://quay.io/repository/biocontainers/perl-autoloader.
