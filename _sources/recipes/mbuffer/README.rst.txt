mbuffer
=======

|downloads|

mbuffer is a tool for buffering data streams with a large set of unique features

======== ===========
Home     http://www.maier-komor.de/mbuffer.html
Versions 20160228
License  GPLv3
Recipe   https://github.com/bioconda/bioconda-recipes/tree/master/recipes/mbuffer
======== ===========

Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install mbuffer

and update with::

   conda update mbuffer



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/mbuffer.
