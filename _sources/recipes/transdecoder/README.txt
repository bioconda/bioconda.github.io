.. _`transdecoder`:

transdecoder
============

|downloads|

TransDecoder identifies candidate coding regions within transcript sequences, such as those generated by de novo RNA-Seq transcript assembly using Trinity, or constructed based on RNA-Seq alignments to the genome using Tophat and Cufflinks.

======== ===========
Home     https://transdecoder.github.io/
Versions 2.1.0
License  Broad Institute
Recipe   https://github.com/bioconda/bioconda-recipes/tree/master/recipes/transdecoder
======== ===========

Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install transdecoder

and update with::

   conda update transdecoder



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/transdecoder.
