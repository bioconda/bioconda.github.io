Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install openbabel

and update with::

   conda update openbabel



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/openbabel.
