bioconductor-annotationdbi
==========================

|downloads|

Provides user interface and database connection code for annotation data packages using SQLite data storage.

======== ===========
Home     http://bioconductor.org/packages/release/bioc/html/AnnotationDbi.html
Versions 1.30.1, 1.32.0, 1.32.2, 1.32.3, 1.34.4, 1.36.0
License  Artistic-2.0
Recipe   https://github.com/bioconda/bioconda-recipes/tree/master/recipes/bioconductor-annotationdbi
======== ===========

Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install bioconductor-annotationdbi

and update with::

   conda update bioconductor-annotationdbi



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/bioconductor-annotationdbi.
