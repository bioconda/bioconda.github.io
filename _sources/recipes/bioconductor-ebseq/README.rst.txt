bioconductor-ebseq
==================

|downloads|

Differential Expression analysis at both gene and isoform level using RNA-seq data

======== ===========
Home     http://bioconductor.org/packages/release/bioc/html/EBSeq.html
Versions 1.12.0, 1.14.0
License  Artistic-2.0
Recipe   https://github.com/bioconda/bioconda-recipes/tree/master/recipes/bioconductor-ebseq
======== ===========

Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install bioconductor-ebseq

and update with::

   conda update bioconductor-ebseq



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/bioconductor-ebseq.
