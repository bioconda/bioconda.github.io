minialign
=========

|downloads|

Fast and accurate alignment tool for PacBio and Nanopore long reads.

======== ===========
Home     https://github.com/ocxtal/minialign
Versions 0.3.1, 0.4.0, 0.4.2
License  MIT
Recipe   https://github.com/bioconda/bioconda-recipes/tree/master/recipes/minialign
======== ===========

Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install minialign

and update with::

   conda update minialign



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/minialign.
