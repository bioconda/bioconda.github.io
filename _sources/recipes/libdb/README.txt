======== ===========

Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install libdb

and update with::

   conda update libdb



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/libdb.
