perl-pdf-api2
=============

Facilitates the creation and modification of PDF files

======== ===========
Home     https://metacpan.org/pod/PDF::API2
Versions 2.025
License  Perl
Recipe   https://github.com/bioconda/bioconda-recipes/tree/master/recipes/perl-pdf-api2

Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install perl-pdf-api2

and update with::

   conda update perl-pdf-api2



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/perl-pdf-api2.
