pytest-mpl
==========

|downloads|

pytest plugin to help with testing figures output from Matplotlib

======== ===========
Home     https://github.com/astrofrog/pytest-mpl
Versions 0.5
License  BSD
Recipe   https://github.com/bioconda/bioconda-recipes/tree/master/recipes/pytest-mpl
======== ===========

Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install pytest-mpl

and update with::

   conda update pytest-mpl



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/pytest-mpl.
