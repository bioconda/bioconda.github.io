Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install scikits-datasmooth

and update with::

   conda update scikits-datasmooth



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/scikits-datasmooth.
