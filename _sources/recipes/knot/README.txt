Detection of knots in protein folds.

Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install knot

and update with::

   conda update knot



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/knot.
