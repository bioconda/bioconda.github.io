.. _`datamash`:

datamash
========

|downloads|

GNU Datamash is a command-line program which performs basic numeric,textual and statistical operations on input textual data files.

======== ===========
Home     http://www.gnu.org/software/datamash
Versions 1.0.6, 1.1.0
License  GPL3
Recipe   https://github.com/bioconda/bioconda-recipes/tree/master/recipes/datamash/1.1.0
======== ===========

Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install datamash

and update with::

   conda update datamash



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/datamash.
