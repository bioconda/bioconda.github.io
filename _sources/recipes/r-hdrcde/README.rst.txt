.. _`r-hdrcde`:

r-hdrcde
========

|downloads|

Computation of highest density regions in one and two dimensions, kernel estimation of univariate density functions conditional on one covariate, and multimodal regression.

======== ===========
Home     http://www.robjhyndman.com/software/hdrcde
Versions 3.1
License  GPL (>= 2)
Recipe   https://github.com/bioconda/bioconda-recipes/tree/master/recipes/r-hdrcde
======== ===========

Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install r-hdrcde

and update with::

   conda update r-hdrcde



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/r-hdrcde.
