graphclust-wrappers
===================

|downloads|

The set of individual perl wrappers extracted from GraphClust pipeline

======== ===========
Home     http://www.bioinf.uni-freiburg.de/Software/GraphClust/
Versions 0.1, 0.1.10, 0.1.11, 0.1.12, 0.1.7, 0.1.8, 0.1.9
License  GPLv3
Recipe   https://github.com/bioconda/bioconda-recipes/tree/master/recipes/graphclust-wrappers
======== ===========

Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install graphclust-wrappers

and update with::

   conda update graphclust-wrappers



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/graphclust-wrappers.
