autoconf
========

|downloads|

Autoconf is an extensible package of M4 macros that produce shell scripts to automatically configure software source code packages.

======== ===========
Home     http://www.gnu.org/software/autoconf/
Versions 2.69
License  GPL 3
Recipe   https://github.com/bioconda/bioconda-recipes/tree/master/recipes/autoconf
======== ===========

Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install autoconf

and update with::

   conda update autoconf



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/autoconf.
