biogridpy
=========

|downloads|

Python client for the BioGRID REST API webservice

======== ===========
Home     https://github.com/arvkevi/biogridpy
Versions 0.1.1
License  MIT
Recipe   https://github.com/bioconda/bioconda-recipes/tree/master/recipes/biogridpy
======== ===========

Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install biogridpy

and update with::

   conda update biogridpy



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/biogridpy.
