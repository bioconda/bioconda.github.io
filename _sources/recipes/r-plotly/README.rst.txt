.. _`r-plotly`:

r-plotly
========

|downloads|

Easily translate ggplot2 graphs to an interactive web-based version and/or create custom web-based visualizations directly from R. Once uploaded to a plotly account, plotly graphs (and the data behind them) can be viewed and modified in a web browser.

======== ===========
Home     https://plot.ly/r, https://github.com/ropensci/plotly
Versions 2.0.16, 4.5.2
License  MIT + file LICENSE
Recipe   https://github.com/bioconda/bioconda-recipes/tree/master/recipes/r-plotly
======== ===========

Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install r-plotly

and update with::

   conda update r-plotly



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/r-plotly.
