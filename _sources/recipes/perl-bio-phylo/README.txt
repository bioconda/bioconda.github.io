perl-bio-phylo
==============

|downloads|

An object-oriented Perl toolkit for analyzing and manipulating phyloinformatic data.

======== ===========
Home     http://biophylo.blogspot.com/
Versions 0.58
License  perl_5
Recipe   https://github.com/bioconda/bioconda-recipes/tree/master/recipes/perl-bio-phylo
======== ===========

Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install perl-bio-phylo

and update with::

   conda update perl-bio-phylo



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/perl-bio-phylo.
