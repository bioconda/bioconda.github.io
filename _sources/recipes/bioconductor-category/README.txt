bioconductor-category

A collection of tools for performing category analysis.

Versions 2.36.0, 2.38.0
License  Artistic-2.0
Recipe   https://github.com/bioconda/bioconda-recipes/tree/master/recipes/bioconductor-category
======== ===========

Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install bioconductor-category

and update with::

   conda update bioconductor-category

A Docker container is available at https://quay.io/repository/biocontainers/bioconductor-category.
