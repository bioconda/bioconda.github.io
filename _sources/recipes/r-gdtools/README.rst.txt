r-gdtools

Useful tools for writing vector graphics devices.

Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install r-gdtools

and update with::

   conda update r-gdtools



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/r-gdtools.
