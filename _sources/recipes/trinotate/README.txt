trinotate
=========

|downloads|

Trinotate is a comprehensive annotation suite designed for automatic functional annotation of transcriptomes, particularly de novo assembled transcriptomes, from model or non-model organisms

======== ===========
Home     https://trinotate.github.io/
Versions 3.0.1
License  
Recipe   https://github.com/bioconda/bioconda-recipes/tree/master/recipes/trinotate
======== ===========

Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install trinotate

and update with::

   conda update trinotate



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/trinotate.
