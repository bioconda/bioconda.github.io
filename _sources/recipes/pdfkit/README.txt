.. _`pdfkit`:

pdfkit
======

|downloads|

Wkhtmltopdf python wrapper to convert html to pdf using the webkit rendering engine and qt

======== ===========
Home     https://pypi.python.org/pypi/pdfkit
Versions 0.5.0
License  MIT License
Recipe   https://github.com/bioconda/bioconda-recipes/tree/master/recipes/pdfkit
======== ===========

Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install pdfkit

and update with::

   conda update pdfkit



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/pdfkit.
