nonpareil
=========

|downloads|

Estimate average coverage and create Nonpareil curves for metagenomic datasets.

======== ===========
Home     https://github.com/lmrodriguezr/nonpareil
Versions 2.4.01
License  Artistic License 2.0
Recipe   https://github.com/bioconda/bioconda-recipes/tree/master/recipes/nonpareil

Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install nonpareil

and update with::

   conda update nonpareil

A Docker container is available at https://quay.io/repository/biocontainers/nonpareil.
