.. _`r-dpeak`:

r-dpeak
=======

|downloads|

This package provides functions for fitting dPeak, a statistical framework to deconvolve ChIP-seq peaks.

======== ===========
Home     http://dongjunchung.github.io/dpeak/
Versions 2.0.1
License  GPL (>= 2)
Recipe   https://github.com/bioconda/bioconda-recipes/tree/master/recipes/r-dpeak
======== ===========

Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install r-dpeak

and update with::

   conda update r-dpeak



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/r-dpeak.
