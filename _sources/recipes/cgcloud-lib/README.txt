Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install cgcloud-lib

and update with::

   conda update cgcloud-lib



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/cgcloud-lib.
