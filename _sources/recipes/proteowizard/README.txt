Tools for dealing with mass spectrometry files (e.g., mzML, mzXML, mzIdentML, MGF)

Versions 3_0_9992

Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install proteowizard

and update with::

   conda update proteowizard
