.. _`ggplot`:

ggplot
======

|downloads|

An extremely un-pythonic package for doing exactly what ggplot2 does

======== ===========
Home     https://github.com/yhat/ggplot/
Versions 0.6.8
License  BSD
Recipe   https://github.com/bioconda/bioconda-recipes/tree/master/recipes/ggplot
======== ===========

Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install ggplot

and update with::

   conda update ggplot



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/ggplot.
