fermi2
======

|downloads|

Fermi2 focuses on the exploration of FMD-index as a graph.

======== ===========

Versions r188

Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install fermi2

and update with::

   conda update fermi2
