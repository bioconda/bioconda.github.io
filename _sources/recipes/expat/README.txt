.. _`expat`:

expat
=====

|downloads|

Expat is an XML parser library written in C. It is a stream-oriented parser in which an application registers handlers for things the parser might find in the XML document (like start tags).

======== ===========
Home     http://www.libexpat.org/
Versions 2.1.0
License  MIT
Recipe   https://github.com/bioconda/bioconda-recipes/tree/master/recipes/expat/2.1.0
======== ===========

Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install expat

and update with::

   conda update expat



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/expat.
