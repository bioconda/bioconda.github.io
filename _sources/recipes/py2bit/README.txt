py2bit
======

|downloads|

A package for accessing 2bit files using lib2bit

======== ===========
Home     https://github.com/dpryan79/py2bit
Versions 0.1.0, 0.2.0
License  MIT
Recipe   https://github.com/bioconda/bioconda-recipes/tree/master/recipes/py2bit
======== ===========

Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install py2bit

and update with::

   conda update py2bit

A Docker container is available at https://quay.io/repository/biocontainers/py2bit.
