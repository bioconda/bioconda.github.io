Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install megahit

and update with::

   conda update megahit



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/megahit.
