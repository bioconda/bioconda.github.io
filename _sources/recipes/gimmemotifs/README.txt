Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install gimmemotifs

and update with::

   conda update gimmemotifs



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/gimmemotifs.
