Various R programming tools for model fitting.

Versions 2.16.2
License  GPL-2
Recipe   https://github.com/bioconda/bioconda-recipes/tree/master/recipes/r-gmodels
======== ===========

Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install r-gmodels

and update with::

   conda update r-gmodels



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/r-gmodels.
