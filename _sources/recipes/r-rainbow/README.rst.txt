r-rainbow
=========

|downloads|

Functions and data sets for functional data display and outlier detection.

======== ===========
Home     https://sites.google.com/site/hanlinshangswebsite/
Versions 3.3
License  GPL (>= 2)
Recipe   https://github.com/bioconda/bioconda-recipes/tree/master/recipes/r-rainbow
======== ===========

Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install r-rainbow

and update with::

   conda update r-rainbow



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/r-rainbow.
