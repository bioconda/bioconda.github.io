Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install biobambam

and update with::

   conda update biobambam



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/biobambam.
