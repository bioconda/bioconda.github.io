Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install r-ade4

and update with::

   conda update r-ade4



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/r-ade4.
