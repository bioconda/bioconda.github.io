ucsc-chainsplit
===============

|downloads|

Split chains up by target or query sequence

======== ===========
Home     http://hgdownload.cse.ucsc.edu/admin/exe/
Versions 324, 332
License  varies; see http://genome.ucsc.edu/license
Recipe   https://github.com/bioconda/bioconda-recipes/tree/master/recipes/ucsc-chainsplit
======== ===========

Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install ucsc-chainsplit

and update with::

   conda update ucsc-chainsplit



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/ucsc-chainsplit.
