.. _`r-tiff`:

r-tiff
======

|downloads|

This package provides an easy and simple way to read, write and display bitmap images stored in the TIFF format. It can read and write both files and in-memory raw vectors.

======== ===========
Home     http://www.rforge.net/tiff/
Versions 0.1.5
License  GPL-2 | GPL-3
Recipe   https://github.com/bioconda/bioconda-recipes/tree/master/recipes/r-tiff
======== ===========

Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install r-tiff

and update with::

   conda update r-tiff



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/r-tiff.
