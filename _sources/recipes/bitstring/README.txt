bitstring
=========

|downloads|

Simple construction, analysis and modification of binary data.

======== ===========
Home     https://github.com/scott-griffiths/bitstring
Versions 3.1.5
License  MIT License
Recipe   https://github.com/bioconda/bioconda-recipes/tree/master/recipes/bitstring
======== ===========

Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install bitstring

and update with::

   conda update bitstring



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/bitstring.
