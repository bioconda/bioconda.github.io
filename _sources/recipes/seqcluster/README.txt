Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install seqcluster

and update with::

   conda update seqcluster



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/seqcluster.
