r-readr
=======

|downloads|

Read flat/tabular text files from disk (or a connection).

======== ===========
Home     https://github.com/hadley/readr
Versions 0.2.2
License  GPL (>= 2)
Recipe   https://github.com/bioconda/bioconda-recipes/tree/master/recipes/r-readr
======== ===========

Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install r-readr

and update with::

   conda update r-readr



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/r-readr.
