r-changepoint
=============

|downloads|

Implements various mainstream and specialised changepoint methods for finding single and multiple changepoints within data.  Many popular non-parametric and frequentist methods are included.  The cpt.mean(), cpt.var(), cpt.meanvar() functions should be your first point of call.

======== ===========
Home     https://github.com/rkillick/changepoint/
Versions 2.2.1
License  GPL
Recipe   https://github.com/bioconda/bioconda-recipes/tree/master/recipes/r-changepoint
======== ===========

Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install r-changepoint

and update with::

   conda update r-changepoint
