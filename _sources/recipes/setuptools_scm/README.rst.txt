setuptools_scm
==============

|downloads|

the blessed package to manage your versions by scm tags

License  MIT License
Recipe   https://github.com/bioconda/bioconda-recipes/tree/master/recipes/setuptools_scm
======== ===========

Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install setuptools_scm

and update with::

   conda update setuptools_scm



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/setuptools_scm.
