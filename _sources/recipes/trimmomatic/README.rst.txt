trimmomatic
===========

|downloads|

A flexible read trimming tool for Illumina NGS data

======== ===========
Home     http://www.usadellab.org/cms/?page=trimmomatic
Versions 0.35, 0.36
License  GPLv3
Recipe   https://github.com/bioconda/bioconda-recipes/tree/master/recipes/trimmomatic
======== ===========

Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install trimmomatic

and update with::

   conda update trimmomatic



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/trimmomatic.
