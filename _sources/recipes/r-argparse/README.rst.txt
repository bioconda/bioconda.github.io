.. _`r-argparse`:

r-argparse
==========

|downloads|

A command line parser to be used with Rscript to write "#!" shebang scripts that gracefully accept positional and optional arguments and automatically generate usage.

======== ===========
Home     https://github.com/trevorld/argparse
Versions 1.0.1
License  GPL (>= 2)
Recipe   https://github.com/bioconda/bioconda-recipes/tree/master/recipes/r-argparse
======== ===========

Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install r-argparse

and update with::

   conda update r-argparse



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/r-argparse.
