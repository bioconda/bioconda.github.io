.. _`r-sets`:

r-sets
======

|downloads|

Data structures and basic operations for ordinary sets, generalizations such as fuzzy sets, multisets, and fuzzy multisets, customizable sets, and intervals.

======== ===========
Home     
Versions 1.0_16
License  GPL-2
Recipe   https://github.com/bioconda/bioconda-recipes/tree/master/recipes/r-sets
======== ===========

Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install r-sets

and update with::

   conda update r-sets



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/r-sets.
