clark
=====

|downloads|

Fast, accurate and versatile k-mer based classification system

======== ===========
Home     http://clark.cs.ucr.edu/
Versions 1.2.2_b, 1.2.3
License  GPL3
Recipe   https://github.com/bioconda/bioconda-recipes/tree/master/recipes/clark
======== ===========

Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install clark

and update with::

   conda update clark



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/clark.
