gqt

GQT is a genotype query interface.

======== ===========
Home     https://github.com/ryanlayer/gqt
Versions 1.1.3
License  MIT
Recipe   https://github.com/bioconda/bioconda-recipes/tree/master/recipes/gqt
======== ===========

Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install gqt

and update with::

   conda update gqt



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/gqt.
