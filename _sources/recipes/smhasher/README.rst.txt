Python extension for smhasher hash functions

Versions 0.150.1
License  MIT License
Recipe   https://github.com/bioconda/bioconda-recipes/tree/master/recipes/smhasher
======== ===========

Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install smhasher

and update with::

   conda update smhasher



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/smhasher.
