Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install cnvkit

and update with::

   conda update cnvkit



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/cnvkit.
