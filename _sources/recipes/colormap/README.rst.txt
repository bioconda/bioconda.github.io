colormap
========

|downloads|

Utilities to ease manipulation of matplotlib colormaps and color codecs (e.g., hex2rgb)

Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install colormap

and update with::

   conda update colormap
