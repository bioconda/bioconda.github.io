r-hwriter

Easy-to-use and versatile functions to output R objects in HTML format

Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install r-hwriter

and update with::

   conda update r-hwriter
