.. _`bioconductor-affyio`:

bioconductor-affyio
===================

|downloads|

Routines for parsing Affymetrix data files based upon file format information. Primary focus is on accessing the CEL and CDF file formats.

======== ===========
Home     http://bioconductor.org/packages/release/bioc/html/affyio.html
Versions 1.40.0, 1.42.0
License  LGPL (>= 2)
Recipe   https://github.com/bioconda/bioconda-recipes/tree/master/recipes/bioconductor-affyio
======== ===========

Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install bioconductor-affyio

and update with::

   conda update bioconductor-affyio



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/bioconductor-affyio.
