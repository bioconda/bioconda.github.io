Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install zifa

and update with::

   conda update zifa



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/zifa.
