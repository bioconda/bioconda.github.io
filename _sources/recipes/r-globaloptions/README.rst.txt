.. _`r-globaloptions`:

r-globaloptions
===============

|downloads|

It provides more controls on the option values such as validation and filtering on the values, making options invisible or private.

======== ===========
Home     https://github.com/jokergoo/GlobalOptions
Versions 0.0.8
License  GPL (>= 2)
Recipe   https://github.com/bioconda/bioconda-recipes/tree/master/recipes/r-globaloptions
======== ===========

Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install r-globaloptions

and update with::

   conda update r-globaloptions



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/r-globaloptions.
