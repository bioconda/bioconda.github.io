flask-bower
===========

======== ===========

Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install flask-bower

and update with::

   conda update flask-bower



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/flask-bower.
