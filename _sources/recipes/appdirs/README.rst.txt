appdirs
=======

|downloads|

A small Python module for determining appropriate platform-specific dirs, e.g. a "user data dir".

======== ===========
Home     http://github.com/ActiveState/appdirs
Versions 1.4.0
License  MIT License
Recipe   https://github.com/bioconda/bioconda-recipes/tree/master/recipes/appdirs
======== ===========

Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install appdirs

and update with::

   conda update appdirs



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/appdirs.
