hmmer
=====

|downloads|

Biosequence analysis using profile hidden Markov models

======== ===========
Home     http://hmmer.janelia.org/
Versions 2.3.2, 3.1b2
License  GPL3
Recipe   https://github.com/bioconda/bioconda-recipes/tree/master/recipes/hmmer
======== ===========

Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install hmmer

and update with::

   conda update hmmer



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/hmmer.
