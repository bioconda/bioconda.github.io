brewer2mpl
==========

|downloads|

Connect colorbrewer2.org color maps to Python and matplotlib

======== ===========
Home     https://github.com/jiffyclub/brewer2mpl/wiki
Versions 1.4.1
License  MIT License
Recipe   https://github.com/bioconda/bioconda-recipes/tree/master/recipes/brewer2mpl
======== ===========

Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install brewer2mpl

and update with::

   conda update brewer2mpl



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/brewer2mpl.
