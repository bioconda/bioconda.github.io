bioconductor-bsgenome.mmusculus.ucsc.mm9
========================================

|downloads|

Full genome sequences for Mus musculus (Mouse) as provided by UCSC (mm9, Jul. 2007) and stored in Biostrings objects.

======== ===========
Home     http://bioconductor.org/packages/release/data/annotation/html/BSgenome.Mmusculus.UCSC.mm9.html
Versions 1.4.0
License  Artistic-2.0
Recipe   https://github.com/bioconda/bioconda-recipes/tree/master/recipes/bioconductor-bsgenome.mmusculus.ucsc.mm9
======== ===========

Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install bioconductor-bsgenome.mmusculus.ucsc.mm9

and update with::

   conda update bioconductor-bsgenome.mmusculus.ucsc.mm9



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/bioconductor-bsgenome.mmusculus.ucsc.mm9.
