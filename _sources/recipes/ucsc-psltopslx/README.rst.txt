ucsc-psltopslx
==============

|downloads|

Convert from psl to pslx format, which includes sequences

======== ===========
Home     http://hgdownload.cse.ucsc.edu/admin/exe/
Versions 324, 332
License  varies; see http://genome.ucsc.edu/license
Recipe   https://github.com/bioconda/bioconda-recipes/tree/master/recipes/ucsc-psltopslx
======== ===========

Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install ucsc-psltopslx

and update with::

   conda update ucsc-psltopslx



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/ucsc-psltopslx.
