python-weblogo
==============

|downloads|

Tool for creating sequence logos from biological sequence alignments

======== ===========
Home     http://code.google.com/p/weblogo/
Versions 3.4
License  Public domain
Recipe   https://github.com/bioconda/bioconda-recipes/tree/master/recipes/python-weblogo
======== ===========

Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install python-weblogo

and update with::

   conda update python-weblogo



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/python-weblogo.
