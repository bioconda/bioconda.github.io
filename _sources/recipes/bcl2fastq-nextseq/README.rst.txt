Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install bcl2fastq-nextseq

and update with::

   conda update bcl2fastq-nextseq



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/bcl2fastq-nextseq.
