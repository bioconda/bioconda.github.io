License  perl_5

Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install perl-text-soundex

and update with::

   conda update perl-text-soundex

A Docker container is available at https://quay.io/repository/biocontainers/perl-text-soundex.
