r-knitr
=======

|downloads|

Provides a general-purpose tool for dynamic report generation in R using Literate Programming techniques.

======== ===========
Home     http://yihui.name/knitr/
Versions 1.12.3
License  GPL-2 | GPL-3
Recipe   https://github.com/bioconda/bioconda-recipes/tree/master/recipes/r-knitr
======== ===========

Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install r-knitr

and update with::

   conda update r-knitr



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/r-knitr.
