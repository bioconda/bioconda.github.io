Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install ucsc-autodtd

and update with::

   conda update ucsc-autodtd



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/ucsc-autodtd.
