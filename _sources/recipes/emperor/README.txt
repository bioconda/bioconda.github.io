Recipe   https://github.com/bioconda/bioconda-recipes/tree/master/recipes/emperor
======== ===========

Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install emperor

and update with::

   conda update emperor



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/emperor.
