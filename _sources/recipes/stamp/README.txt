stamp
=====

|downloads|

A graphical software package for analyzing taxonomic and functional profiles.

======== ===========
Home     http://pypi.python.org/pypi/stamp/
Versions 2.1.3
License  Custom
Recipe   https://github.com/bioconda/bioconda-recipes/tree/master/recipes/stamp
======== ===========

Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install stamp

and update with::

   conda update stamp



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/stamp.
