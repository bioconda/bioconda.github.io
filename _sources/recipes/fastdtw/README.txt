fastdtw
=======

|downloads|

Dynamic Time Warping (DTW) algorithm with an O(N) time and memory complexity

======== ===========
Home     https://github.com/slaypni/fastdtw
Versions 0.2.0
License  MIT License
Recipe   https://github.com/bioconda/bioconda-recipes/tree/master/recipes/fastdtw
======== ===========

Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install fastdtw

and update with::

   conda update fastdtw

A Docker container is available at https://quay.io/repository/biocontainers/fastdtw.
