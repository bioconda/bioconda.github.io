Perl extension for using OpenSSL

======== ===========
Home     http://metacpan.org/pod/Net-SSLeay
Versions 1.72, 1.74
License  perl_5
Recipe   https://github.com/bioconda/bioconda-recipes/tree/master/recipes/perl-net-ssleay
======== ===========

Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install perl-net-ssleay

and update with::

   conda update perl-net-ssleay



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/perl-net-ssleay.
