.. _`r-rsm`:

r-rsm
=====

|downloads|

Provides functions to generate response-surface designs, fit first- and second-order response-surface models, make surface plots, obtain the path of steepest ascent, and do canonical analysis.

======== ===========
Home     
Versions 2.7_4
License  GPL-2
Recipe   https://github.com/bioconda/bioconda-recipes/tree/master/recipes/r-rsm
======== ===========

Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install r-rsm

and update with::

   conda update r-rsm



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/r-rsm.
