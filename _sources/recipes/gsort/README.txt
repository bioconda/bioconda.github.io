gsort
=====

|downloads|

sort genomic files according to a genomefile

======== ===========
Home     https://github.com/brentp/gsort
Versions 0.0.1, 0.0.2
License  MIT
Recipe   https://github.com/bioconda/bioconda-recipes/tree/master/recipes/gsort
======== ===========

Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install gsort

and update with::

   conda update gsort



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/gsort.
