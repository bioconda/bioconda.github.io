Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install hisat2

and update with::

   conda update hisat2



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/hisat2.
