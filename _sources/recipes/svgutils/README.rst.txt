.. _`svgutils`:

svgutils
========

|downloads|

Python SVG editor that allows to automatically create publication ready composite SVG figures.

======== ===========
Home     http://neuroscience.telenczuk.pl
Versions 0.1.0
License  MIT License
Recipe   https://github.com/bioconda/bioconda-recipes/tree/master/recipes/svgutils
======== ===========

Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install svgutils

and update with::

   conda update svgutils



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/svgutils.
