avro-python3
============

|downloads|

Avro is a serialization and RPC framework.

======== ===========
Home     http://avro.apache.org/
Versions 1.8.0, 1.8.1
License  Apache License 2.0
Recipe   https://github.com/bioconda/bioconda-recipes/tree/master/recipes/avro-python3
======== ===========

Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install avro-python3

and update with::

   conda update avro-python3



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/avro-python3.
